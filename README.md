# HospitalSys
基于springboot的医院信息管理系统

演示视频：https://live.csdn.net/v/175547

实现的功能：

用户登录、门诊管理、住院管理、系统管理、统计管理、数据中心、排班管理、仓库管理、药房管理等。

用到的技术：

后端：JAVA语言、SpringBoot框架、MySQL数据库、maven依赖管理等；

前端：HTML页面、layUI框架等。
